# ☕ STARBUCKS

스타벅스 랜딩 페이지(홈페이지 + signin)를 만든 예제로 아래 명시된 기술을 활용했습니다.

- 오픈 그래프(The Open Graph protocol)
- 트위터 카드(Twitter Cards)
- 구글 폰트 / 머터리얼 아이콘
- GSAP / ScrollToPlugin
- Swiper
- Youtube API
- ScrollMagic
- Lodash
- Netlify
